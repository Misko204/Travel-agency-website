# Travel Agency Website



A responsive bilingual travel agency web presentation built with HTML, CSS, JavaScript and Bootstrap.



## Features

- Bilingual support (Serbian and English)

- 20+ pages with uniform visual identity

- Dynamic drop-down navigation menu

- Browse travel packages by group (Europe, Summer 2025, Winter 2025)

- Sort and search arrangements by name and price

- Booking system with session storage

- Ratings system for visited destinations

- Photo and video gallery

- Interactive map with agency location

- Responsive design — works on desktop, tablet and mobile



## Destinations

Europe: Prague, Berlin, Budapest

Summer 2025: Hurghada, Kusadasi, Marrakech

Winter 2025: Bansko, Kopaonik, Les Menuires



## Technologies

- HTML5

- CSS3

- JavaScript

- Bootstrap



## How to run

Just open `index.html` in any browser — no installation needed.
